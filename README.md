# React Users

## Overview

This project is a web application built with React that displays a table of users. It includes features such as state management with Zustand, API integration and pagination.

## Features

1. React & Vite

    - The project is built using React with Vite for fast development and optimized builds.

2. State Management with Zustand

    - Zustand is used for global state management, providing a lightweight and easy-to-use alternative to Redux.

3. Pagination
    - The table is paginated to handle large datasets efficiently.

## Installation & Setup

```
# Clone the repository
git clone https://github.com/BadOfficer/React-Users.git
cd React-Users

# Install dependencies
npm install

# Start the json-server
npm run start:server

# Start the development server
npm run dev

# Open your browser and visit:
http://localhost:5173
```
